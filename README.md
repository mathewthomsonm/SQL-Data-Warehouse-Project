# 🏗️ Data Architecture

This project follows the *Medallion Architecture*, consisting of three layers:

## 🥉 Bronze Layer
- Stores raw data exactly as received from source systems.
- Imports ERP and CRM CSV files into SQL Server.
- Preserves the original source data.

## 🥈 Silver Layer
- Cleans, standardizes, and transforms the raw data.
- Handles missing values, duplicates, and data quality issues.
- Creates a consistent data model for analysis.

## 🥇 Gold Layer
- Stores business-ready analytical data.
- Builds fact and dimension tables using a Star Schema.
- Optimized for reporting and dashboard creation.

# 🚀 Project Requirements

## Building the Data Warehouse (Data Engineering)

### Objective

Develop a modern SQL Server Data Warehouse that consolidates ERP and CRM data into a single source of truth for reporting and analytics.

### Specifications

- Import ERP and CRM data from CSV files.
- Clean and standardize the data.
- Integrate multiple sources into one analytical model.
- Use only the latest dataset.
